Galaxy Warrior is a complete Python Pygame game project that can be opened and run in VS Code. It is designed as a larger practical project for Computer Graphics, Programming, or Multimedia subjects.

## Features
- Start menu
- Player spaceship movement
- Enemy waves
- Bullet shooting system
- Collision detection
- Score tracking
- Health bar and lives
- Power-ups
- Multiple levels
- Boss fight
- Game over screen
- Victory screen

## Tools and Technologies

- VS Code

## How to Run in VS Code
1. Install Python.
2. Open this project folder in VS Code.
3. Open terminal in VS Code.
4. Install pygame:
   ```bash
   pip install -r requirements.txt
   ```
5. Run the game:
   ```bash
   python main.py
   ```

## Controls
- Move: Arrow keys or W A S D
- Shoot: Spacebar
- Restart after game over or win: R
- Start game from menu: Enter

## Project Explanation
- `main.py` starts the game.
- `game.py` manages menus, levels, game loop, HUD, and screen states.
- `entities.py` contains classes like Player, Enemy, Boss, Bullet, Star, Explosion, and PowerUp.
- `settings.py` stores all reusable configuration values.
