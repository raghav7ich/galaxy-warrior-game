# VIVA QUESTIONS AND ANSWERS

## 1. What is Pygame?
Pygame is a Python library used for creating games and multimedia applications.

## 2. What is a game loop?
A game loop is the continuous cycle in which the game handles input, updates objects, and redraws the screen.

## 3. What is collision detection?
Collision detection is the process of checking whether two objects have touched each other.

## 4. What is FPS?
FPS means frames per second. It shows how many times the screen updates in one second.

## 5. Why did you use Python for this project?
Python is easy to learn, easy to write, and works well with Pygame for graphical projects.

## 6. What is object-oriented programming in this project?
Object-oriented programming means the game objects are created as classes like Player, Enemy, Boss, and Bullet.

## 7. What is the role of settings.py?
It stores reusable constants like screen size, colors, speed, and FPS values.

## 8. What happens when the player loses all lives?
The game state changes to the game over screen.

## 9. What are power-ups?
Power-ups are special objects collected by the player to get temporary advantages like shield or rapid fire.

## 10. What graphics concepts are used here?
2D drawing, animation, event handling, collision detection, HUD display, and real-time rendering are used.
